
# BitByte Technology

### Add .env file in root folder
``` 
DATABASE_URL: ""
PORT: 6100
JWT_SECRET = ""
JWT_EXPIRE_IN = ""
```

#### To run project
```
npm i
npx prisma generate
npm run start:dev
```
## API Reference

After successfully running the project. You can see api reference -
```
http://localhost:6100/api-docs
```
